Calendar-Availability by pixmin

PHP script used to fetch the data from Google Calendar and show it on a calendar webpage

The page is showing a calendar with several months and the day's details, whether busy, free, or else, and if required the details of the projects' name can also be shown.

Dependencies

PHP and a Google Calendar

Install

Download the files, edit the settings_inc.php config file with your preferences and your Google Calendar Feed details, upload to a server with PHP, and that should be it.
